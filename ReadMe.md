## 📖 Overview

Windows Toolkit is a portable Windows maintenance and diagnostic utility written entirely in **Batch** and **PowerShell**.

It provides a modern console interface with tools for:

- Windows Repair
- Hardware Information
- Software Inventory
- System Maintenance
- Network Repair
- Windows Utilities
- HTML System Report Generation

No installation required.
